# commentpage

##What is happened
This demonstration project comments on the backbone and node. Mongo used as the database.

##Install
```bash
npm install
bower install
```
after you must install [mongodb](https://docs.mongodb.com/v3.2/installation/)

##Run
```bash
mongodb
node server
```
